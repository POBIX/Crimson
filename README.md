# Crimson
A 2D game engine written in C# with modern OpenGL.

# Features
- Entity-Component system similar to Unity's
- Graphics: dynamic lighting, particle system, textures, sprites
- Animations: cameras, tweening, sprite animations
- Shaders: materials, compute shaders, SSBOs, framebuffers, post-processing
- OpenGLSharp: automatically generated C# OpenGL bindings
- Physics: extensible collision system, raycasting, swept AABBs
- Tilemaps: load Tiled tilemaps and tilesets
- Text Rendering: FreeType Unicode font rendering
- Audio: using PortAudio and libsndfile
- Input: keyboard, mouse, gamepad support, actions similar to Godot's
- Dear ImGUI integration
- Human readable C# scene files
